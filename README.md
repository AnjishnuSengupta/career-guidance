## 🌟 Overview

CareerCompass is a comprehensive career guidance platform designed specifically for Class 11 & 12 students, helping them make informed decisions about their academic and professional futures. Our platform offers personalized recommendations based on students' interests, aptitudes, and goals.

![CareerCompass Preview](https://via.placeholder.com/800x400?text=CareerCompass+Preview)

## ✨ Features

### 🧠 Interactive Career Quiz
Take our personalized assessment to discover career paths that match your unique interests and strengths.

### 📚 Academic Stream Explorer
Detailed information about Science, Commerce, and Humanities streams, including subjects, career opportunities, top colleges, and entrance exams.

### 🛣️ Career Path Visualization
Step-by-step roadmaps showing the journey from choosing a stream to becoming a professional in various fields.

### 📑 Resource Hub
Access guides, articles, videos, and downloadable materials to help prepare for entrance exams, explore colleges, and assess aptitude.

### 🤝 Career Counseling
Connect with career counselors for personalized guidance and support.

### 🌓 Dark/Light Mode
Seamless theme switching for comfortable viewing day or night.

## 🚀 Getting Started

### Prerequisites

- Node.js (v18 or newer)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/yourusername/career-compass.git
cd career-compass
```

2. Install dependencies:
```bash
npm install
# or
yarn install
```

3. Run the development server:
```bash
npm run dev
# or
yarn dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser to see the application
